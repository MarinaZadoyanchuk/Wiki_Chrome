Wiki_Chrome
===========
Для встановлення розширення достатньо завантажити файл clue_for_the_wiki.crx .
Для початку спробуйте відкрити даний файл за допомогою Google Chrome, можливо, установка пройде таким чином, але е у всіх так спрацьовує.
Якщо перший варіант установки не спрацював, зайдіть в браузері на chrome://extensions/ і з папки, в якій знаходиться файл clue_for_the_wiki.crx, перетягніть його у вкладку chrome://extensions/ і підтвердіть встановлення.

Дякую, що тестуєте мою програмку))!
